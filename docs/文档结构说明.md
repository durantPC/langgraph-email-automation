## 🎯 文档分类说明

### 1. 快速开始类文档

**目标用户**: 新用户、首次使用者

**文档列表**:
- **使用文档.md** - 简单精炼，快速了解系统功能和使用方法
- **快速开始.md** - 详细的配置步骤，5分钟快速上手

**使用建议**: 
- 第一次使用 → 先看"使用文档.md"
- 需要详细配置步骤 → 再看"快速开始.md"

---

### 2. 配置类文档

**目标用户**: 需要配置系统的用户

**文档列表**:
- **配置说明.md** - 通用配置（QQ邮箱授权码、环境变量等）
- **硅基流动配置指南.md** - 硅基流动API的详细配置步骤
- **模型选择说明.md** - 如何选择、切换AI模型

**使用建议**:
- 基础配置 → "配置说明.md"
- API配置 → "硅基流动配置指南.md"
- 模型选择 → "模型选择说明.md"

---

### 3. 使用类文档

**目标用户**: 已配置好系统，需要了解如何使用

**文档列表**:
- **运行模式说明.md** - 三种运行模式（单次/持续/API）的详细说明

**使用建议**:
- 了解如何运行系统 → "运行模式说明.md"

---

### 4. 技术类文档

**目标用户**: 开发者、运维人员、需要深入了解系统的用户

**文档列表**:
- **说明文档.md** - 完整的技术文档，包含：
  - 系统架构
  - 工作流程
  - API接口
  - 数据库结构
  - 部署指南
  - 故障排查

**使用建议**:
- 需要深入了解系统 → "说明文档.md"
- 部署生产环境 → "说明文档.md" 的部署章节
- 排查问题 → "说明文档.md" 的故障排查章节

---

### 5. 历史类文档

**目标用户**: 需要了解项目历史、变更记录的用户

**文档列表**:
- **变更摘要.md** - 重大变更记录（从Gmail到QQ邮箱等）
- **更新日志.md** - 版本更新历史
- **MIGRATION_COMPLETE.md** - 迁移到国内服务的完成报告

**使用建议**:
- 了解项目变更历史 → "变更摘要.md"
- 查看版本更新 → "更新日志.md"
- 了解迁移过程 → "MIGRATION_COMPLETE.md"

## 📝 文档维护原则

### 文档更新

1. **新增功能** → 更新"使用文档.md"和"说明文档.md"
2. **配置变更** → 更新相关配置文档
3. **版本发布** → 更新"更新日志.md"
4. **重大变更** → 更新"变更摘要.md"

### 文档一致性

- 所有文档顶部添加导航链接
- 保持文档间的交叉引用
- 统一术语和格式

### 文档分类

- 简单文档 → 快速开始类
- 详细文档 → 技术类
- 配置相关 → 配置类
- 历史记录 → 历史类
